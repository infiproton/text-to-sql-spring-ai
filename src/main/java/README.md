# Text-to-SQL Udemy Course Repository

This repository is organized to support a Udemy course on building a Text-to-SQL application. For each module's lessons (where code is applicable), a separate branch has been created. Each branch is named clearly to indicate its purpose and the lesson/module it corresponds to.
